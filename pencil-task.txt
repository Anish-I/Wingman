Use the Pencil MCP tools to design the Wingman app screens.

1. First call get_editor_state to check if Pencil desktop is connected
2. If connected, call get_guidelines with topic='mobile-app'
3. Open or create a document called Wingman-Screens.pen using open_document
4. Build 4 mobile app frames using batch_design with these specs:
   - Frame 1 "Signup": dark background #0f0f1a, purple accent #6c63ff, teal #00d4d4, Pip pigeon mascot centered, Google SSO button, Apple SSO button, heading "Welcome to Wingman"
   - Frame 2 "Chat": dark background, purple gradient user chat bubbles, glassmorphism Pip response bubbles, Pip avatar in header, suggestion chip row at bottom
   - Frame 3 "Apps": dark background, horizontal category filter pills, app grid cards with Connect buttons, green Connected badges
   - Frame 4 "Workflows": dark background, stats bar showing Runs/Active/Success rate, glassmorphism workflow cards, purple floating action button
5. Export or screenshot each frame
6. Report all tool calls made and results

If get_editor_state fails with a connection error, report the exact error message.
